# Reflection 目录说明

本目录用于存放课程期末要求中的小组分工记录和个人反思材料。根据 Lesson 27 & 28 的要求，本目录重点说明：

1. 小组成员在项目中的具体分工。
2. 每位成员对本项目的个人收获、反思和后续学习计划。
3. 个人贡献如何对应 GitHub 仓库中的具体文件、报告和图表。
