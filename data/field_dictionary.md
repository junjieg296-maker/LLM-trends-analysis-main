# 数据字段字典

本文档说明 `data/raw/lens-llm-agents-raw.csv` 与 `data/processed/cleaned_data.csv` 中主要字段的含义和用途。

| 字段 | 含义 | 主要用途 | 备注 |
|---|---|---|---|
| Lens ID | Lens.org 记录标识 | 数据追踪、去重辅助 | 原始数据中的唯一记录线索 |
| Title | 文献标题 | 标题词频、人工核验、Title 去重 | 清洗后覆盖率 100% |
| Publication Date | 出版日期 | 时间识别 | 部分记录可能只有年份 |
| Publication Year | 出版年份 | 年度发文趋势分析 | 清洗后覆盖率 100% |
| Publication Type | 文献类型 | 期刊 / 会议论文分布 | 用于描述样本文献结构 |
| Source Title | 来源期刊或会议 | 来源分布、代表性来源说明 | 存在不同大小写和缩写 |
| Publisher | 出版方 | 辅助描述数据来源 | 非核心分析字段 |
| Source Country | 来源国家 / 地区 | 国家或区域分布分析 | 可能存在缺失 |
| Author/s | 作者 | 作者统计、作者合作网络 | 需过滤空值和无效作者名 |
| Abstract | 摘要 | 主题解释、人工核验 | 清洗后覆盖率约 98.28% |
| Citing Works Count | 被引次数 | 影响力指标、高被引论文表 | 来自 Lens.org |
| Fields of Study | 研究领域 | 学科分布分析 | 多值字段 |
| Keywords | 关键词 | 关键词共现网络 | 单篇内先标准化和去重 |
| References | 参考文献 | 引用网络、共被引网络 | 覆盖率约 81.99% |
| DOI | 数字对象标识符 | DOI 去重、文献溯源 | 清洗后缺失率约 0.19% |
| Open Access | 是否开放获取 | OA 比例分析 | 依赖 Lens.org 标识 |

## 字段质量摘要

| 字段 | 缺失率 | 后续使用 |
|---|---:|---|
| Title | 0.00% | 趋势解释、去重、主题识别 |
| Author/s | 0.38% | 作者统计和合作网络 |
| Abstract | 1.72% | 语义解释和人工核验 |
| References | 18.01% | 引用网络和共被引网络 |
| DOI | 0.19% | 去重与溯源 |

## 说明

字段名称可能因 Lens.org 导出格式存在轻微差异。实际脚本中会先对字段名进行清理，再执行去重、统计和图表生成。
