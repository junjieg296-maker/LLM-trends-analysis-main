# Presentation 目录说明

本目录用于记录课程答辩材料的位置和使用说明。

## 答辩材料

| 文件 | 说明 |
|---|---|
| `LLM_multi_agent_week13_defense_expanded.pptx` | 13 页扩展版答辩 PPT，覆盖 RQ、数据口径、3 图 1 表、核心发现、局限、GitHub 路径和分工 |

## 讲述重点

正式答辩建议围绕以下顺序展开：

1. 研究主题与背景。
2. Research Questions。
3. 数据来源、检索式与筛选流程。
4. 3 图 1 表与核心发现。
5. GitHub 复现路径。
6. 小组分工、个人贡献与局限。

## 备份说明
